pdpy

pdpy is a Pure Data scripting language written in Python.

Things you can do with pdpy:

  • Make Pure Data patches from Python code

  • Use pdpy-lang to generate Pure Data patches

  • Translate Pure Data files from the .pd format to other formats:

    • json

    • xml

    • pickle

Note

This project is under active development. PRs are welcome!
