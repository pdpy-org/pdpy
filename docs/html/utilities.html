Translator class

class pdpy.classes.translator.Translator(json)[source]

Bases: Base

This class maintains and translates a pdpy Obj in memory.

This class loads a file in .pd or .json formats and keeps an internal mirror (aka, translation) between the two. The direction of the translation depends on the input file type. Use the save_* functions to write translations to disk. Alternatively, you can load a .pkl (aka, pickle) file containing a pdpy object.

A dictionary of arguments with the following keys: to: the target format. Can be json, xml, pd, or pkl. fro: the source format. Can be json, xml, pd, or pkl. input: An input file Path, will be formated using pathlib.Path output: An output file Path, will be formated using pathlib.Path encoding (str, default is ‘utf-8’): Encoding of the input file source (str, inferred from input_file): Source file type reflect (bool): If set to True, performs a reflected translation
